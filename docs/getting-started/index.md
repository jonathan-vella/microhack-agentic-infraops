# Getting Started

Choose your entry point based on where you are in the workshop journey:

## Before the Event

| Task | Where |
|---|---|
| Verify you can participate | [Readiness Gate](setup.md#readiness-gate) |
| Install tools and configure environment | [Environment Setup](setup.md#prerequisites) |
| Create your working repository | [Setup Steps](setup.md#setup-steps) |
| Read the business scenario | [Workshop Prep](workshop-prep.md) |

## First 10 Minutes on Event Day

| Task | Where |
|---|---|
| Confirm Dev Container is running | [Setup Guide](setup.md#dev-container) |
| Authenticate to Azure | [Setup Steps](setup.md#setup-steps) |
| Verify custom agents are visible | [Readiness Gate](setup.md#readiness-gate) item 6 |
| Start Challenge 1 | [Challenge 1: Requirements](../challenges/challenge-1-requirements.md) |

## During the Workshop

| Need | Where |
|---|---|
| Challenge instructions | [Challenges](../challenges/index.md) |
| Agent and prompting help | [Copilot Guide](../guides/copilot-guide.md) |
| Quick command reference | [Quick Reference Card](../guides/quick-reference-card.md) |
| Something broke | [Troubleshooting](../reference/troubleshooting.md) |

## After the Event

| Task | Where |
|---|---|
| Clean up Azure resources | [Post-Event Cleanup](setup.md#post-event-cleanup) |
| Provide feedback | [Feedback](../about/feedback.md) |
